Enable automatic sync so tasks update in real-time when you move them to "In Progress".

1 Create a GitHub Personal Access Token
  1. Go to github.com/settings/tokens/new
  2. Name: "Task Manager Sync"
  3. Expiration: 90 days (or No expiration)
  4. Check: repo (Full control of private repositories)
  5. Click "Generate token"
  6. Copy the token (starts with ghp_ or github_pat_)
2 Paste your token here Your token is stored locally in your browser and never sent anywhere except GitHub.
